# Staircase Pro Modern

Обновлённая версия проекта `staircase.zip`.

Что изменено:
- сохранён основной функционал исходного проекта;
- обновлён интерфейс всех основных разделов;
- добавлен анимированный фон и более современная визуальная система;
- добавлена вкладка `Портфолио` с блоком о мастере, фото, опытом, достижениями и примерами работ;
- сохранены калькулятор, авторизация, личный кабинет, мои заказы и админ-раздел.

## Запуск

```bash
npm install
npm run dev
```

## Переменные окружения

Скопируйте `.env.example` в `.env.local` и заполните значения Supabase.

## Примечание

Если переменные окружения не заполнены, интерфейс откроется, но сетевые операции Supabase работать не будут.
